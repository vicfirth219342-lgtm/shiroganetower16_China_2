## 使い方

1. 上記のプロンプトをコピー
2. `【 】` をすべて物件情報に書き換え
3. Claude Codeの最初のメッセージとして貼り付け
4. 不足情報があればClaude Codeが質問してくれる

## カスタマイズポイント

### 物件タイプ別の追記例

**低層マンション（タワーでない場合）**
```
タワーマンションの訴求は控えめに。
代わりに「低層の静寂」「プライベート感」「閑静な住環境」を訴求してください。
```

**郊外・緑豊かなエリア**
```
都心へのアクセスより、自然環境・生活の豊かさを優先して訴求してください。
```

**リノベーション物件**
```
築年数よりリノベーションの内容・クオリティを前面に出してください。
「新築のように住める」という訴求軸でコピーを考えてください。
```

**投資向け（インカムゲイン訴求）**
```
実需だけでなく、賃貸需要・利回り・流動性も訴求ポイントに含めてください。
```
